# Microsoft Agent 365 Observability Extensions - Semantic Kernel

This package provides Semantic Kernel observability and tracing extensions for Microsoft Agent 365.

## Features

- **Semantic Kernel Integration**: Seamless integration with Semantic Kernel applications
- **Agentic Tracing**: Specialized tracing for AI agent workflows using Semantic Kernel
- **OpenTelemetry Support**: Built-in OpenTelemetry tracing for standardized observability
- **Kairo Semantic Convention**: Follows Kairo specifications for consistent telemetry

## Quick Start

### Import required functionalities
```python
from microsoft_kairo import configure
from microsoft_kairo.observability.wrappers.semantic_kernel import KairoInstrumentorSemanticKernel
```

### Basic Usage

Configure your connection and integrate Semantic Kernel tracing:
```python
# Configure your connection to Kairo
status = configure(
    service_name="kairo-semantic-kernel-wrapper",
    service_namespace="kairo-semantic-kernel-wrapper-test",
    agent_id="sample-agent-01"
)

# Integrate Semantic Kernel's tracing with Kairo
KairoInstrumentorSemanticKernel().instrument()
```

This integrates all incoming traces (logs) generated by the agentic framework using Semantic Kernel into Kairo following the Kairo Semantic Convention specification.

## Installation

```bash
pip install microsoft-agents-a365-observability-extensions-semantic-kernel
```

## Requirements

- Python 3.9+
- Semantic Kernel >= 1.0.0
- OpenTelemetry API >= 1.20.0
