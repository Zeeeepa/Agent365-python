# Microsoft Agent 365 Observability Extensions - OpenAI Agents

This package provides OpenAI Agents SDK observability and tracing extensions for Microsoft Agent 365.

## Features

- **OpenAI Agents Integration**: Seamless integration with OpenAI Agents SDK applications
- **Agentic Tracing**: Specialized tracing for AI agent workflows using OpenAI Agents SDK
- **OpenTelemetry Support**: Built-in OpenTelemetry tracing for standardized observability
- **Kairo Semantic Convention**: Follows Kairo specifications for consistent telemetry

## Quick Start

### Import required functionalities
```python
from microsoft_kairo import configure
from microsoft_kairo.observability.wrappers.openai_agents import KairoInstrumentorOpenAIAgents
```

### Basic Usage

Configure your connection and integrate OpenAI Agents tracing:
```python
# Configure your connection to Kairo
status = configure(
    service_name="kairo-openaiAgents-wrapper",
    service_namespace="kairo-openaiAgents-wrapper-test",
    agent_id="sample-agent-01"
)

# Integrate OpenAI Agents SDK's tracing with Kairo
KairoInstrumentorOpenAIAgents().instrument()
```

This integrates all incoming traces (logs) generated by the agentic framework using OpenAI Agents SDK into Kairo following the Kairo Semantic Convention specification.

## Installation

```bash
pip install microsoft-agents-a365-observability-extensions-openai
```

## Requirements

- Python 3.9+
- OpenAI Agents >= 0.2.6
- OpenTelemetry API >= 1.20.0
