# Kairo Tooling OpenAI

OpenAI Agents SDK integration tools for Kairo AI agent tooling.

## Description

This package provides integration tools and utilities specifically for OpenAI Agents SDK, enabling observability and tooling capabilities for OpenAI-based AI agents.

## Installation

```bash
pip install kairo-tooling-openai
```

## Dependencies

- microsoft-agents-a365-tooling >= 0.1.0
- openai-agents
- asyncio-throttle

## Features

- OpenAI Agents SDK integration
- MCP tool registration service
- Tooling utilities for OpenAI-based applications
