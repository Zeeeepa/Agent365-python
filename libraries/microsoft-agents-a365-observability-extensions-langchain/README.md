# Microsoft Agent 365 Observability Extensions - LangChain

This package provides LangChain observability and tracing extensions for Microsoft Agent 365.

## Features

- **LangChain Integration**: Seamless integration with LangChain applications
- **Agentic Tracing**: Specialized tracing for AI agent workflows using LangChain
- **OpenTelemetry Support**: Built-in OpenTelemetry tracing for standardized observability
- **Kairo Semantic Convention**: Follows Kairo specifications for consistent telemetry

## Quick Start

### Import required functionalities
```python
from microsoft_kairo import configure
from microsoft_kairo.observability.wrappers.langchain import KairoInstrumentorLangChain
```

### Basic Usage

Configure your connection and integrate LangChain tracing:
```python
# Configure your connection to Kairo
status = configure(
    service_name="kairo-langchain-wrapper",
    service_namespace="kairo-langchain-wrapper-test",
    agent_id="sample-agent-01"
)

# Integrate LangChain's tracing with Kairo
KairoInstrumentorLangChain().instrument()
```

This integrates all incoming traces (logs) generated by the agentic framework using LangChain into Kairo following the Kairo Semantic Convention specification.

## Installation

```bash
pip install microsoft-agents-a365-observability-extensions-langchain
```

## Requirements

- Python 3.9+
- LangChain >= 0.1.0
- OpenTelemetry API >= 1.20.0
