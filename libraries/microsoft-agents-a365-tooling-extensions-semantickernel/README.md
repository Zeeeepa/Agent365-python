# Kairo Tooling Semantic Kernel

Microsoft Semantic Kernel integration tools for Kairo AI agent tooling.

## Description

This package provides integration tools and utilities specifically for Microsoft Semantic Kernel framework, enabling observability and tooling capabilities for Semantic Kernel-based AI agents.

## Installation

```bash
pip install kairo-tooling-semantic-kernel
```

## Dependencies

- microsoft-agents-a365-tooling >= 0.1.0
- semantic-kernel >= 1.0.0
- aiohttp >= 3.8.0

## Features

- Semantic Kernel agent integration
- Observability and tracing capabilities
- Tooling utilities for SK-based applications
