# Kairo Tooling Azure Foundry

Azure AI Foundry integration tools for Kairo AI agent tooling.

## Description

This package provides integration tools and utilities specifically for Azure AI Foundry (Azure AI Projects and Agents), enabling tooling capabilities for Azure AI-based applications.

## Installation

```bash
pip install kairo-tooling-azurefoundry
```

## Dependencies

- microsoft-agents-a365-tooling >= 0.1.0
- azure-ai-projects >= 1.0.0
- azure-ai-agents >= 1.1.0b4
- azure-identity >= 1.12.0

## Features

- Azure AI Foundry integration
- Azure AI Projects and Agents support
- Tooling utilities for Azure AI applications
